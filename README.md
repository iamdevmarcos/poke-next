## Running locally

1. Clone this repo:

```sh
$ git clone https://github.com/iamdevmarcos/poke-next
```

2. Then go to the project's folder:

```sh
cd poke-next
```

3. Install all dependencies:

```sh
npm install
```

4. Run locally:

```sh
npm run dev
```
